# Finite Automata Simulator

A simple, interactive web-based simulator for Finite Automata. This tool allows users to visualize and simulate state machines directly in the browser. 

## Live Demo

You can try out the live version of the project here:
👉 **[FA Simulator Demo](https://fasimulator.vercel.app/)**

## Usage
Simply open `index.html` in your browser, or visit the live demo link above to start using the simulator.
